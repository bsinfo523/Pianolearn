Mikrofon aktivieren

Das Mikrofon braucht HTTPS – als lokale Datei blockiert Android den Zugriff.

So geht es:
1 Auf dem PC: Python installieren (python.org) falls nicht vorhanden
2 start_server.py im selben Ordner wie pianolearn.html ausführen
3 Handy und PC ins selbe WLAN – die angezeigte IP-Adresse in Chrome öffnen, z.B.:
https://192.168.1.42:8443/pianolearn.html
4 Chrome zeigt Sicherheitswarnung → Erweitert → Weiter klicken
✓ Jetzt auf 🎤 Mikrofon tippen → Erlaubnis erteilen → fertig!
💡 Einfachste Lösung: Play Store → "HTTP Server - html developer" von Palapa installieren → HTTPS aktivieren → pianolearn.html öffnen. Fertig!

🎹 Alternativ: USB/Bluetooth MIDI-Keyboard → 🎹 MIDI-Button – kein HTTPS nötig.
